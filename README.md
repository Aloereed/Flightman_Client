本项目为数据库与实现课程的航班管理系统的客户端。  
编程语言：C++（with Qt 5.14.2）  
兼容的数据库环境:MySQL 5.7  
兼容的操作系统：Windows（x64）、Android（ARM、ARM64）、macOS  
关于Android端下的MySQL插件编译可移步[为Qt5 Android构建MySQL驱动](https://gitee.com/aloereed/Qt_Android_MySQL_Plugin)
